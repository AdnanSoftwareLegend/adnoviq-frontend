# ADNOVIQ

ADNOVIQ is a modern AI-powered storefront built with Next.js, TypeScript, Tailwind CSS, and an Express.js backend.

## Features
- Product catalog and details page
- Cart experience with persistent storage
- Explore page with search and filtering
- Authentication and role-based dashboard pages
- AI product description generator

## Run locally
```bash
cd frontend
npm install
npm run dev
```

```bash
cd backend
npm install
npm run dev
```
